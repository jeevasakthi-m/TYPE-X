## 🎮 Features

- 🕒 Real-time tracking of:
  - ⏱️ WPM (Words Per Minute)
  - ✅ Accuracy %
  - 🔄 Timer countdown
- 🎯 Random quote generation for each test
- 🔁 Restart functionality for practice rounds
- 📱 Responsive UI for mobile & desktop
- 🧼 Clean, minimal design for better focus
